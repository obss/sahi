# Ultralytics Models

SAHI integrates with [Ultralytics YOLO26](https://docs.ultralytics.com/models/yolo26)
along with YOLO11, YOLOv8, and all other Ultralytics model variants (detection,
segmentation, and oriented bounding boxes).
