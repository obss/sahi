# Ultralytics Models

SAHI integrates with [Ultralytics YOLO26](https://docs.ultralytics.com/models/yolo26),
[Ultralytics YOLO11](https://docs.ultralytics.com/models/yolo11), and
[Ultralytics YOLOv8](https://docs.ultralytics.com/models/yolov8), along with all
other Ultralytics model variants (detection, segmentation, and oriented bounding
boxes).
